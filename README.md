# 🌳 Árvore Genealógica — Família Silva · Angelo · Oliveira · Thiago

Sistema pessoal de pesquisa e visualização genealógica, cobrindo 9 gerações da família de **Vinicius de Oliveira Angelo** (RJ, 1998).

## 📱 Visualização

### Desktop
Abra [`index.html`](index.html) — visualização completa com zoom, pan e edição.

### Mobile (recomendado no celular) ⭐
Abra [`mobile.html`](mobile.html) — otimizado para toque, com as famílias **totalmente separadas em abas**:

- 🔵 **Paterno** — Jorge Fernando · Nilcea · Xavier · Souza · Aguiar
- 🟣 **Materno** — Patricia · Oliveira · Guimarães · Thiago Outeiro 🇵🇹
- 👤 **Você & Filhas** — Vinicius · Jade · Clara

## 🗂️ Estrutura

```

├── index.html              # Visualizador desktop (zoom/pan)
├── mobile.html             # Visualizador mobile (abas, separado)

├── app.js                  # Lógica desktop

```

## 🚀 GitHub Pages

Para ver no celular de qualquer lugar, ative **GitHub Pages** nas configurações do repositório:

1. `Settings` → `Pages`
2. Source: `Deploy from a branch` → `main` / `root`
3. Abra: `https://highlevelvnc.github.io/thre2/mobile.html`

## 📝 Como Editar

- **Desktop**: abra `index.html`, clique em qualquer pessoa para editar
- **Mobile**: `mobile.html` é somente-leitura — para editar use o desktop e re-exporte o JSON
- Os dados ficam em `arvore_atualizada.json` (editar direto também funciona)
